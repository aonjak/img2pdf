# แปลงรูปเป็น PDF (Image → PDF)

เครื่องมือแปลงรูปภาพเป็นไฟล์ PDF ที่ทำงาน **ในเครื่องล้วนๆ ไม่ส่งรูปขึ้นเน็ต** — เลือกหลายรูป, ปรับคุณภาพ, ย่อขนาด, เลือกขนาดหน้า (พอดีรูป/A4/Letter) แล้วได้ PDF คมๆ ไฟล์เล็ก

👉 **ใช้งาน:** เปิด `index.html` ในเบราว์เซอร์ หรือเปิดผ่าน GitHub Pages

## ฟีเจอร์
- เลือกรูปได้หลายรูป เรียงลำดับ/ลบได้
- ปรับคุณภาพ 50–100%
- ย่อขนาดด้านยาวสุด (เต็ม / 3000 / 2400 / 2000 / 1600 / 1200 px)
- ขนาดหน้า PDF: พอดีกับรูป / A4 / Letter (จัดแนวนอน–ตั้งอัตโนมัติ)
- ฝัง JPEG ตรงเข้า PDF (DCTDecode) → คมและไฟล์เล็ก
- ออฟไลน์ได้ ไฟล์เดียวจบ ไม่ต้องลงอะไรเพิ่ม

## ทำเป็นแอป / APK

2. หรือเอาลิงก์ไปสร้าง APK ฟรีที่ [PWABuilder](https://www.pwabuilder.com)
